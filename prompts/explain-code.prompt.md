# Explain Code

Explain the following code in detail at **${input:depth}** level.

## Structure your explanation as:

1. **Overview** (1-2 sentences): what does this code do at a high level?
2. **Flow Diagram**: Mermaid flowchart showing the execution flow
3. **Step-by-Step**: walk through the code line by line (or block by block for shallow depth)
4. **Key Concepts**: explain any patterns, algorithms, or language features used
5. **Complexity**: time and space complexity if applicable
6. **Potential Issues**: bugs, edge cases, or improvements worth noting

Adjust depth based on the level requested:
- **shallow**: overview + key concepts only
- **medium**: overview + flow + block-by-block explanation
- **deep**: everything including line-by-line analysis and complexity
